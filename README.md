# Тренажер английских слов

## О проекте
Простой консольный тренажер для изучения английских слов. Программа показывает английское слово, а пользователь вводит русский перевод.

## Функциональность
- Случайный показ английских слов
- Проверка правильности перевода
- Подсчет правильных ответов
- Статистика результатов

## Установка и запуск
```bash
git clone https://github.com/Khripko228/eng.train
cd english-trainer

python -m venv venv

# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

python main.py
